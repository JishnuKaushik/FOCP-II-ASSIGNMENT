## 🔥 Featured Projects

### 🤖 AI Roast Generator

A console-based interactive program that generates humorous roast messages.

**Highlights:**

* Randomized output using `rand()`
* Dynamic placeholder replacement `{name}`
* Continuous user interaction loop
* Clean modular structure

---

### 🎓 Student Excuse Generator

A creative console application that generates random excuses for assignments.

**Highlights:**

* Multiple excuse templates stored in `vector<string>`
* Random selection with proper seeding (`srand(time(0))`)
* Accurate placeholder replacement
* Beginner-friendly and clean logic

## 🧠 How I Used AI (Deep Insight)

AI was used as a **development assistant**, not just for generating code.

### 🔹 Prompt Engineering

* Began with simple prompts
* Iteratively refined them for accuracy and better output
* Controlled logic and structure through detailed instructions

### 🔹 Code Development

* Generated initial program structure
* Improved modularity using functions
* Enhanced readability with proper formatting

### 🔹 Debugging Process

AI helped identify and fix real issues:

**Bug:** Same output generated every run
**Cause:** Missing random seed
**Fix:**
`srand(time(0));`
**Result:** Different outputs on each execution

---

### 🔹 Optimization Improvements

* Replaced `cin` with `getline()` for better input handling
* Fixed placeholder replacement logic using loops
* Removed encoding issues caused by unsupported characters

---

## 🐞 Bug Fixing Summary

| Issue                              | Cause               | Fix                        |
| ---------------------------------- | ------------------- | -------------------------- |
| Same output every run              | Missing random seed | `srand(time(0))`           |
| Placeholder not replaced correctly | Incorrect logic     | Loop-based replacement     |
| Weird terminal characters          | Encoding mismatch   | Removed special characters |
